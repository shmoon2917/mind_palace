# 노드 라이브러리 정리

### NPM vs NPX

NPM 과 NPX는 라이브러리가 아니다.

- `npm` 은 Node.js 의 **의존성**과 **패키지 관리**를 위한 패키지 매니저

- `npx` 는 Node 패키지를 실행시키는 하나의 도구이며 **설치하지 않고도** 무거운 패키지와 같은 것들을 **오직 실행만 시킬 수 있는** 도구!
